# SHOPPING-CART

## Getting Started
After installing the project and start mongod you can use the helper homepage at:
- [API helper Homepage](http://localhost:3000/api)

## Prerequisites
Project have dependencies that require:
- node v8.9.4
- mongo v3.6.1
- npm 5.6.0

Project require running mongod

## Installing
```sh
$ npm install
```

## Running the tests
```sh
$ npm run test
```
